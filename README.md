# Citizen Registry

Приложение для работы с картотекой граждан. В проекте есть список граждан, карточка с разделами, редактирование основной информации и дашборд с показателями

## Стек

- React
- TypeScript
- Vite
- React Router
- React Query
- TanStack Table
- Recharts
- CSS Modules
- Express

## Возможности

- Дашборд с ключевыми показателями и графиками
- Таблица граждан с поиском, фильтрами, сортировкой и пагинацией
- Экспорт отфильтрованного списка граждан в CSV
- Карточка гражданина с вкладками: основная информация, документы, семья, образование и занятость
- Редактирование основной информации гражданина
- Обработка состояний загрузки, ошибки и пустых списков

## Страницы

- `/` — дашборд
- `/citizens` — картотека граждан
- `/citizens/:id` — карточка гражданина

## Запуск

Команды выполняются из корня проекта

Backend запускается на `http://localhost:3001`

```bash
cd backend
npm install
npm run dev
```

Frontend запускается отдельно

```bash
cd frontend
npm install
npm run dev
```

## Проверка

```bash
cd frontend
npm run lint
npm run build
```
